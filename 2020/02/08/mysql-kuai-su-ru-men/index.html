MYSQL

快速学习mysql

（注：我们们初级的web安全学习者只需要明白简单的SQL操作即可，以下我就介绍一下基本操作问题）

首先明白你们需要学那些

你们作为web初学者，强烈建议你们只需要会基本数据库操作即可。作为一个当初傻不愣登的学完了mysql全部操作的人的忠告。你们现阶段只要会数据库的打开，数据库的增删改，数据的查询即可。

具体操作请参考在群里发的，里面有各种细化

怎么进入数据库

我们可以打开cmd在cmd中输入以下命令（需要你先设置好系统路径）

有两种启动方式：

一种是先输入用户名，然后再输入密码。

#我的用户名是root，如果你的是别的请自己更改替换成你的用户名
mysql -uroot -p

另一种就是直接启动

#-proot的意思是password root,这里你需要换成你自己的密码。
mysql -uroot -proot

还有就是可以通过辅助工具连接上数据库，这里就不在讲述了。

数据库的基本操作

连接数据库showdatabases;#查询所有库
use库名;#选择进入数据库
showtables;#查询库下的所有表
createdatabase库名 [charset字符集];#创建数据库
dropdatabase库名;#删除数据库

简单操作演示

#创建一个字符集为utf8的数据库
create database test3 character set utf8;
#创建一张表，并设置字段格式
create table test30( id  int(10) not null, name  char(10) not null , sex char(5) not null );

表的基本操作

#创建表
create teable 表名（                                                                     列名数据类型,                                                                              .....                                                                                   ） engine myisam charset utf8;
desc表名;#显示表的属性、属性类型、主键信息，是否为NULL等信息
showcolumnsfrom表名;#同上
droptable表名;#删除表
renametable oldname to newname;#更改表名
select * from表名;#查看表的数据内容
truncate表名;#删除表，再重新建表；与delete的区别是delete是相对行而言，是把表内的数据擦除

添加表的数据：
insertinto 表名( 需要添加的列)  values (需要添加的数据);
修改表中的数据：
update表名set要修改的列where判断表达式;
删除表中的数据：
deletefrom表名where判断表达式;

showcreatetable表名;#查看表的建表语句
insertinto库名.表名(导入的表) select选择要从表中导入的内容from库名.表名(被导入的表);
#把一张表的内容，导入到另一张表。

简单操作

insert into test30 (id,name,sex) value (1,'名字','man');

列的操作

在不同位置添加列：
altertable表名add列名称列类型列参数;#添加在列的最后
altertable表名add列名称列类型列参数after某列;#添加到指定列的后面
altertable表名add列名称列类型列参数first;#添加到第一列
altertable表名drop列名;#删除指定列
altertable表名modify列名新列类型新参数;#修改列的数据类型
altertable表名change旧列名新列名新列类型新列参数;#修改列名和列类型

数据类型

整型：tinyint、smallint、mediuint、int、bigint

int系列的声明时的参数：(M)：指定宽度，需要与zerofill连用，unsigned：声明无符号，zerofill：如果为空值，用“0”填充。

浮点型：float(M,D)、decimal(M,D)M：为“精度”，代表总位数，D：为“标度”，代表小数。

字符型：char(M)：定长类型varchar(M)：变长类型M：为字符宽度

text：文本类型

bolb：二进制类型，用于存储图片、音频等信息。

查询

select*from 表名 where在字段1=值 1or 字段2=值2；

select*from 表名 where 字段1=值1 and 字段2=值2；

select*from 表名 where 查询值 between 值1 and 值2

通配符： _ (代表任意一个字符) %（代表0….N个字符）

select*from 表名 where name like ‘___’;(五个)

select 列名 from 表名；

select max（字段1），min（字段2） from 表；

Count() 统计指定列表不为null的记录行数

max（） 计算指定列的最大值，如果指定列是字符串类型，那么使用字符串排序运算。

Min ( ) 计算指定列的最小值，如果指定列是字符串类型，那么使用字符串排序运算。

Sum( ) 计算指定列的数值和，如果指定列类型不是数值类型，那么计算结果为0.

Avg( ) 计算指定列的平均值，如果指定列类型不为数值类型，那么计算结果为0.

书写顺序

select 字段，聚合函数，from 表名 where 初筛条件 group by 要分组的字段 having 还要筛选的值 order by 参数 顺序 limit 要几条

select id from test30 order by id desc；

select * from test10 limit 1,2;
